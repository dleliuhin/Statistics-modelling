Требуется определить, используя метод статистического моделирования, какое минимальное количество ЗЧ необходимо, чтобы вероятность безотказной работы (ВБР) системы за время T была не менее P0. ЛФРС определяется из структурно-надежностной схемы системы, которая также как и остальные исходные данные для решения задачи приведены ниже повариантно. Номер варианта выбирает преподаватель. Количество реализаций случайной величины N для получения результатов с требуемой точностью необходимо обосновать.
